# simple-instagram

## Project Overview
This project is a simple clone of Instagram, demonstrating basic functionalities of social media applications.

## Features
- User authentication.
- Post creation and sharing.
- Profile management.

## Technologies Used
- Node.js
- Express
- MongoDB

## Usage
1. Clone the repository.
2. Install dependencies.
3. Run the application.

## Future Plans
- Add real-time features using WebSockets.
- Improve the UI/UX design.
